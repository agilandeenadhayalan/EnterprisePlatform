# M06: Containerization

Learn multi-stage Dockerfiles, Docker Compose orchestration, health checks, and container networking.

## Key Concepts

- **Multi-Stage Builds** — Minimize image size, separate build/runtime
- **Compose Orchestration** — Service dependencies, startup order, networking
- **Health Checks** — Container health monitoring, restart policies
- **Volume Management** — Named volumes, bind mounts, data persistence

## Running the Module

```bash
python -m learning.phase_1.src.m06_containerization.demo
```

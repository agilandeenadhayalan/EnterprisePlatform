# M03: RESTful API Design

Learn Richardson Maturity Model, HATEOAS, cursor-based pagination, and idempotency patterns.

## Key Concepts

- **Richardson Maturity** — Level 0 through Level 3 REST APIs
- **Pagination** — Cursor-based vs offset, keyset pagination
- **Idempotency** — Idempotency keys, safe retries, at-most-once semantics
- **Error Handling** — Problem Details (RFC 7807), consistent error responses

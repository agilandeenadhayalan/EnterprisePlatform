# M04: Database Connection Patterns

Learn connection pooling, the Repository pattern, N+1 query detection, and transaction management.

## Key Concepts

- **Connection Pooling** — Pool sizing, health checks, connection lifecycle
- **Repository Pattern** — Abstracting data access, unit of work
- **N+1 Queries** — Detection, eager loading, dataloader pattern
- **Transactions** — ACID, isolation levels, optimistic locking

python -m learning.phase_1.src.m04_database_patterns.demo

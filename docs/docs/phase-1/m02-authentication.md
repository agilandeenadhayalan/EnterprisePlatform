# M02: Authentication & Authorization

Learn JWT lifecycle management, OAuth2 flows, role-based access control, and secure token rotation.

## Key Concepts

- **JWT Tokens** — Access/refresh token pairs, claims, expiration
- **OAuth2 Flows** — Authorization code, client credentials, PKCE
- **RBAC** — Roles, permissions, policy evaluation
- **Token Rotation** — Refresh token rotation, revocation, device binding

```bash
python -m learning.phase_1.src.m02_authentication.demo
```

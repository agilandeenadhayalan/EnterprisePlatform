# M05: Caching Strategies

Learn cache-aside, write-through, cache stampede prevention, and TTL strategies.

## Key Concepts

- **Cache-Aside** — Read-through pattern, lazy population
- **Write-Through** — Synchronous cache updates on writes
- **Stampede Prevention** — Probabilistic early expiration, lock-based refresh
- **TTL Strategies** — Time-based, event-based invalidation
